## Fallyx Behaviours
> python backend scripts for processing pdf+excel files and uploading to the backend

### Usage
For each home:
- find the respective home folder for the scripts
- place pdf and spreadsheet files into `./[home]/downloads`
- run `./[home]/run_script.py`

### Homes
Berkshire
- different format from millcreek & oneill
- has follow up notes

Millcreek 
- almost identical to oneill code
- has follow up notes

Oneill
- almost identical to millcreek
- has follow up notes

Banwell
- no follow up notes support
